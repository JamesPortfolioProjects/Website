Water Quality Tester — Full Technical Overview

This water tester was designed to measure pollutants and water quality in real time. We used Arduino with custom pressure designs and SolidWorks for housing. Key challenges included maintaining flow, cost constraints (leading to less precise sensors and size trade-offs), and display simplicity for the user. The output values were aligned with National Health Services benchmarks. I collaborated with a team of three, and if given more time and resources, I would optimize accuracy, add LCD integration, and shrink the design further.

AI Eye Tracker — Full Technical Overview

This project leveraged Python with OpenCV libraries for continuous facial tracking, despite Python’s threading limitations. We used queueing and a custom velocity-smoothing algorithm to create smooth transitions. GPIO requirements were fulfilled using a custom-built foot pedal. Time constraints and a team of three pushed us to innovate on limited resources. The final showcase received extremely positive feedback, especially from children interacting with the tech.

Magic Mirror — Full Technical Overview

Built solo as a holiday project for Alma, this magic mirror uses a Raspberry Pi and an affordable acrylic two-way mirror with custom server communication. A Flutter app allows for remote customization and control. Future improvements include voice or touch controls and professional-grade smoothing. I learned new coding languages and server security practices during this build and plan to continue refining it.
